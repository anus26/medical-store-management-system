# 🏥 Medical Store Management System

A **Django-based Medical Store Management System** designed to efficiently manage medicines, sales, stock, and billing operations. This system helps streamline pharmacy workflows and improves inventory tracking.

---

## 🚀 Features

* 💊 Manage Medicines (Add, Update, Delete)
* 📦 Stock Management with low stock alerts
* 🧾 Sales & Billing System
* 👥 Customer Management
* 📊 Dashboard with analytics (sales, stock, etc.)
* 🔐 User Authentication & Role-based Access
* 🧮 Invoice generation system

---

## 🛠️ Tech Stack

* **Backend:** Django, Python
* **Frontend:** HTML, CSS, Bootstrap
* **Database:** SQLite / PostgreSQL
* **Tools:** Git, VS Code

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/medical-store-management-system.git
cd medical-store-management-system
```

### 2️⃣ Create Virtual Environment

```bash
python -m venv env
source env/bin/activate   # (Linux/Mac)
env\Scripts\activate      # (Windows)
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run Migrations

```bash
python manage.py makemigrations
python manage.py migrate
```

### 5️⃣ Create Superuser

```bash
python manage.py createsuperuser
```

### 6️⃣ Run Server

```bash
python manage.py runserver
```
